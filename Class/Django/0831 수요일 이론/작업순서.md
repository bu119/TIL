

### 가상환경 설치 및 활성화

```bash
$ python -m venv venv
$ source venv/Scripts/activate
```

### 장고 설치

```bash
$ pip install django==3.2.13
또는
$ pip install -r requirements.txt
```

### 프로젝트 생성

```bash
$ django-admin startproject pjt .
```

### 앱생성

```bash
$ python manage.py startapp articles
```

### 어플 등록

```bash
settings.py
```

### 서버 실행

```bash
$ python manage.py runserver
```
